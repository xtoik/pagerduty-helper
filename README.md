# PagerDuty Helper
## Scope
Chrome extension that provides helpful functionalities for this web page.
## Functionalities
### Schedules
- Condensed View: when active, condenses the Schedules view.
##  Features
- Webpack
- Linting
- React
## Install
- Download ZIP for latest release.
- Unpack it in your preferred folder for unpacked extension.
- In Chrome / Chromium / Edge open Extensions > Manage Extensions
- Press the _Load unpacked_ button.
- Select the folder where you unpacked the ZIP file.
